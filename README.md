# EEG-Based ADHD Classification

Bachelor thesis project focused on EEG-based ADHD classification using deep learning models (EEGNet and NeuroGPT) with 80/20 split and Leave-One-Subject-Out (LOSO) evaluation.

## Models
- EEGNet
- NeuroGPT

## Evaluation Protocols
- 80/20 Train/Test Split
- Leave-One-Subject-Out (LOSO)

## Dataset
EEG data for ADHD / Control children  
IEEE DataPort (Nasrabadi et al., 2020)
